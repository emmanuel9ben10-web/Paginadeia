Descripción del Proyecto
Este proyecto es una plataforma web desarrollada en Java enfocada en la enseñanza y divulgación de la Inteligencia Artificial Generativa. La aplicación tiene como finalidad brindar información clara, sencilla e interactiva para que cualquier usuario pueda conocer qué es la IA, cómo funciona y cuáles son sus beneficios en la actualidad sin necesidad de conocimientos previos.🎓

Contexto Académico: Esta plataforma fue desarrollada como proyecto final para la materia de Ingeniería de Software, aplicando estrictamente las fases de planeación, análisis, diseño y desarrollo de sistemas.🎯 

Objetivo GeneralDesarrollar una plataforma web funcional que permita a los usuarios aprender sobre Inteligencia Artificial Generativa mediante contenido educativo estructurado, control de acceso mediante registro de usuarios y evaluaciones interactivas.⚠️

Problemática a ResolverActualmente, existe un gran interés por aprender sobre Inteligencia Artificial; sin embargo, la mayoría de los recursos disponibles en la red presentan los siguientes obstáculos:🌐 Barrera del idioma: Gran parte de la documentación y cursos de calidad están exclusivamente en inglés.🧠

Complejidad alta: Las explicaciones suelen ser difíciles de entender para principiantes.🛠️ Requisitos técnicos: Muchos portales requieren conocimientos avanzados de programación o matemáticas.

Este proyecto rompe esas barreras ofreciendo una alternativa accesible, visual y en español.🛠️ Tecnologías UtilizadasÁreaTecnologíaLenguaje de programaciónJavaDesarrollo WebJSP (JavaServer Pages), HTML5, CSS3Entorno de Desarrollo (IDE)Apache NetBeansBase de DatosMySQLControl de VersionesGit y GitHubPlaneación y GestiónGanttProject⚙️ Funcionalidades Principales👤 Módulo de UsuariosRegistro de usuarios: Creación de cuentas nuevas con almacenamiento seguro.Inicio de sesión: Autenticación de identidad para acceso al contenido.Gestión de perfiles: Espacio personalizado para el usuario.

📚 Módulo de Contenido IAConceptos Básicos: Explicaciones sencillas sobre Inteligencia Artificial Generativa.Recursos Educativos: Material didáctico diseñado para un aprendizaje progresivo.

📝 Módulo de EvaluacionesQuizzes Interactivos: Preguntas dinámicas para poner a prueba lo aprendido.Historial de Calificaciones: Almacenamiento y consulta de los resultados en la base de datos.

🗄️ Base de Datos: Repositorio relacional encargado de estructurar las tablas de Usuarios, Contraseñas, Contenido educativo, Preguntas y Calificaciones.
